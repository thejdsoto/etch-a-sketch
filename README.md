# etch-a-sketch

30 August 2022

Deployment of the program's alpha version into Github. Still lacking many key features. Premature version of the program was rushed and pushed to Github due to boredom and I want to try other projects already. The program will be constantly updated and improved from time to time.

Current features:
- Like a typical etch-a-sketch app, user can choose a color from the color palette and use that color to sketch pixels on the canvass

Lacking features:
- A prompt that will ask the user how many pixels the canvass should be
- Eraser function
- Clear all funciton
- UI improvements
- Web responsiveness/mobile friendly
